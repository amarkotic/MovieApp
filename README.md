# MovieApp
 MovieApp provides comprehensive movies and shows overview. 

 ## Technologies used
• Reactive paradigm - Combine
 
• UI - UIKit + Snapkit

• Dependencies - SPM

• Local database - Realm

• Architecture - CLEAN Architecture by Uncle Bob

• Restful API
